
## Run Locally

Clone the project

```bash
  git clone https://github.com/pupiaguayo/liteflix
```

Go to the project directory

```bash
  cd liteflix
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
